# SvgBadgeGenerator
Reads a .csv file and generates .svg(s) from it, and also puts them all in one file.

# Setup
- have a folder named `csh50th_cards` in your computer's `Downloads` folder. The program will not work otherwise
- run `main.py`

# Notes
- If there are multiple majors, use ` / ` (with the spaces) to separate them. If a comma is present, use `~` instead, the program will replace this with a comma.
